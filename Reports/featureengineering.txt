
Feature engineering:

From the given data the information about how many times a movie is rated is extracted which is included in featureengineering.csv


From the given data number of movies being rated in a given year is generated by covering the timeframe column in to date format. (Featureengineeringdatedata.csv)

Number of movies rated by a particular user which contains a movie tag is included in featureengineeringtags.csv

By having the number of movies being rated in a year we can predict the popularity of the movie
